# 電気設計シミュレーター ポータル

電気工事と電気設計の基礎から応用まで段階的に学習できるWebベースシミュレーターコレクションです。

## 概要

このプロジェクトは、電気工事の学習・訓練および第二種電気工事士試験対策のための包括的なシミュレーター集です。ポータル型のインターフェースにより、設計知識と基礎理論の2つのカテゴリーから適切なシミュレーターを選択できます。全てのシミュレーターは完全レスポンシブ対応で、様々なデバイスで利用可能です。

## シミュレーター構成

### 🎯 設計知識カテゴリー

#### 1. 電気回路シミュレーター（初級）
- **ファイル**: `circuit-simulator-rich.html`
- **概要**: 基本的な電気回路の動作を視覚的に学習
- **主な機能**:
  - 単純スイッチ回路
  - 3路・4路スイッチ回路
  - コンセント・プラグの接続
  - 電流の流れの可視化

#### 2. 住宅分電盤回路シミュレーター（上級）
- **ファイル**: `distribution-board-simulator.html`
- **バージョン**: v8.3
- **概要**: 実際の住宅分電盤を模した総合的なシミュレーター
- **主な機能**:
  - リアルな分電盤レイアウト
  - 主幹・分岐ブレーカー操作
  - 過負荷・短絡シミュレーション
  - 太陽光発電連系対応
  - 回路設計と配線

#### 3. 住宅ブレーカー遮断シミュレーター（中級）
- **ファイル**: `residential-breaker-simulator.html`
- **概要**: 部屋ごとの電気使用量管理とブレーカー遮断条件の学習
- **主な機能**:
  - 部屋別の負荷管理
  - 家電機器の電力計算
  - 漏電ブレーカー動作
  - 感震ブレーカー対応
  - リアルタイムの電力使用状況表示

#### 4. 太陽光発電システムシミュレーター（中級）
- **ファイル**: `solar-power-simulator.html`
- **概要**: 太陽光発電システムの設計・発電量計算・経済性評価
- **主な機能**:
  - パネル配置シミュレーション
  - 発電量予測計算
  - 経済効果分析
  - 専門用語集
  - 都道府県別日照データ

#### 5. 電気工学基礎インタラクティブシミュレーター（中級）
- **ファイル**: `improved-electrical-simulator.html`
- **概要**: 電力・電圧・電流の包括的な理解と電線の許容電流値計算
- **主な機能**:
  - **電力と力率**: 有効電力・無効電力・皮相電力の関係性学習、力率三角形の可視化
  - **電圧降下計算**: 単相・三相回路での電圧降下、距離・電流・抵抗による影響
  - **電線の許容電流**: VVF・IV・CV・CVT各種電線の許容電流値計算、温度補正・布設条件考慮
  - **配電方式比較**: 単相2線式・単相3線式・三相3線式の特徴比較
  - **計算ロジック明示**: ステップバイステップの計算過程、数式と根拠の表示、実用計算例

### 📚 基礎理論カテゴリー

#### 1. オームの法則・基礎理論（基礎）
- **ファイル**: `ohms-law-basic-theory.html`
- **概要**: 電気回路の基本であるオームの法則、直列・並列回路、合成抵抗について詳しく学習
- **主な機能**:
  - オームの法則の実践学習
  - 直列・並列回路のシミュレーション
  - 合成抵抗の自動計算
  - 回路図の視覚的解説
  - 改良されたマルチメーター表示

#### 2. 交流回路シミュレーター（応用）
- **ファイル**: `ac-circuit-simulator.html`
- **概要**: 交流回路の基本概念、インピーダンス、位相差について視覚的に学習
- **主な機能**:
  - 交流波形のリアルタイム描画
  - RLC回路のシミュレーション
  - インピーダンスの計算と視覚化
  - 位相差の動的表示

#### 3. 第二種電気工事士 法令対策（試験対策）
- **ファイル**: `electrical-law-simulator.html`
- **概要**: 第二種電気工事士筆記試験の法令分野を効率的に学習
- **主な機能**:
  - 電気工事士法の重要ポイント
  - 電気設備技術基準の要点
  - 試験頻出の数値・条件
  - 模擬問題とクイズ形式

### 🔧 その他のツール

#### 1. ケーブル種類シミュレーター（参考資料）
- **ファイル**: `cable-types-simulator.html`
- **概要**: 各種電気ケーブルの構造と特性を視覚的に学習
- **主な機能**:
  - VVF、CV、IV等の主要ケーブルの構造図
  - ケーブル選定の基準と用途
  - 許容電流と使用環境の関係
  - ケーブル比較機能

#### 2. 24時間換気シミュレーター（応用）
- **ファイル**: `ventilation-simulator.html`
- **概要**: 建築基準法で定められた24時間換気システムの設計と換気量計算
- **主な機能**:
  - 第1種〜第3種換気方式の動作原理
  - 必要換気量の自動計算
  - 部屋別の換気計画
  - 換気扇の選定支援

## 使用方法

1. `index.html`をWebブラウザで開いてポータルページにアクセスします
2. 「設計知識」または「基礎理論」タブから学習したいカテゴリーを選択します
3. 各カテゴリー内から適切な難易度のシミュレーターを選択します
4. 各シミュレーターには「ポータルへ戻る」ボタンがあり、いつでもメインページに戻れます

### ナビゲーション構造
```
ポータル (index.html)
├── 設計知識タブ
│   ├── 電気回路シミュレーター
│   ├── 住宅分電盤回路シミュレーター
│   ├── 住宅ブレーカー遮断シミュレーター
│   └── 太陽光発電システムシミュレーター
├── 基礎理論タブ
│   ├── オームの法則・基礎理論
│   ├── 交流回路シミュレーター
│   └── 第二種電気工事士 法令対策
└── その他のツール（個別アクセス）
    ├── ケーブル種類シミュレーター
    └── 24時間換気シミュレーター
```
